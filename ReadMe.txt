Данный программный продукт реализован на основе библиотеки CsvHelper.
CsvHelper - библиотека .NET для чтения и записи файлов CVS.
Файл с таблицей, которую нужно отредактировать разместить в папке проекта с именем "InputFile.csv".
Сформированный файл появиться там же (в папке проекта), с названием "OutputFile.csv".

                                                !!!ОЧЕНЬ ВАЖНО!!!
Не обновляйте CsvHelper выше версии 19.0.0, если вы обновите выше этой версии, некоторые функции программы перестанут работать.
